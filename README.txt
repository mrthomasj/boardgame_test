Instruções para compilação do desafio:

O arquivo gameConfig DEVE estar na raiz do executável do desafio (Ex: G:\_Projects\BankruptTest\BankruptTest\bin\Debug\netcoreapp3.1\gameConfig.txt).

Por padrão, no arquivo .zip o gameConfig já estará presente na pasta. Caso contrário não será possível executar o teste, apresentando uma mensagem de erro no console.

Carregar a .sln pelo Visual Studio e compilar o programa através do debugger, via tecla F5.
